## [0.9.0] - 2026-02-07 - Phase 6 Refinements
**Goal:** Complete Narrative Polish & Canon Consistency.

### Added
- **Interactive Tooltips:** Implemented `[[Term::Definition]]` syntax parsing in `build_publication.ps1`.
- **Sidebar Library:** Created and populated `SIDEBAR_LIBRARY.md` with expandable lore terms.
- **New Lore Entries:** Added "Safe Mode", "High-Fidelity Rendering", "High-Fidelity Healing", "Resonance Dysphoria", "The Silence" to Repository.

### Changed
- **Manuscript (Chapters 1-17):**
    - Removed all Exposition `<aside>` blocks to improve narrative flow ("Show, Don't Tell").
    - Replaced exposition with inline context tooltips.
    - Enhanced "Sci-Fi Horror" tone in Chapters 7-17 (Safe Mode visuals, Glitch aesthetics).
    - Updated Taro's character arc to reflect his nature as "The Silence/Void".
- **Repository:**
    - Updated `01_POWER_SYSTEM.md` with System Render Modes (Hi-Fi, Safe Mode, Void).
    - Updated `03_CHARACTERS_REGISTRY.md` to reclassify Taro (Void) and add Elara/Kimura.
    - Updated `the_score_mechanics.md` with visual guides for new render modes.

### Fixed
- **HTML Build:** Fixed `aside` tag artifacts in the sidebar generation.
- **Canon:** Reconciled discrepancies between early chapters and late-stage "Void" reveals.
