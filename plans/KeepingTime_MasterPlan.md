## D9 **[NEW]** — Beat classifier: add `combat` and `impact` types

**Location:** `BEAT_TYPE_VALUES` object and `classifyBeatType()` function.

**What:** The engine currently has `action` and `conflict` beat types but no dedicated `combat` or `impact` type. Fight scenes need their own vol/coh values — higher vol, lower coherence, faster BPM response.

```javascript
// Add to BEAT_TYPE_VALUES:
combat:    [0.80, 0.45],  // active fight beats — high tension, moderate chaos
impact:    [0.90, 0.35],  // hit/sound-word beats — peak tension spike
lockOn:    [0.50, 0.80],  // TARGET_ANALYSIS readouts — tactical, controlled
```

```javascript
// Add to classifyBeatType():
// Sound words on their own line (*CRACK*, *THOOM*, *ZZZTT*, *SHHKK*, etc.)
if (/^<em>(CRACK|THOOM|ZZZTT|SHHKK|PING|SKREEE|ZAP|PANG|PULSE|SHATTER)\.<\/em>$/i.test(el.innerHTML.trim())) return 'impact';

// TARGET_ANALYSIS blocks
if (/TARGET_ANALYSIS:/i.test(text)) return 'lockOn';

// Active fight prose
if (/\b(baton|cage|strike|deflect|dodge|counter|leap|intercept|thrust)\b/i.test(text)) return 'combat';
```
